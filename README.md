# TrendNest E-Commerce

A MERN e-commerce application with premium black and white design.

## Features

- **Product Catalog** - Browse and search products
- **Shopping Cart** - Add, update, and remove items
- **User Accounts** - Profile management and order history
- **Checkout Process** - Secure order placement
- **Order Tracking** - View order history and status

## Pages

- **Home** - Hero section with video background
- **Products** - Product listing with search
- **Product Detail** - Individual product pages
- **Cart** - Shopping cart management
- **Checkout** - Order form and summary
- **Orders** - Order history
- **Account** - User profile and orders

## Design

- Black and white color scheme
- Font Awesome icons
- Responsive layout
- Smooth animations and transitions
- Minimalist typography

## Technology

- React
- React Router
- Tailwind CSS
- Font Awesome
- Custom hooks for state management
